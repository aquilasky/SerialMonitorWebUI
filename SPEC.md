# SerialMonitor - WebUI串口调试助手规格说明

## 1. 项目概述

### 项目名称
SerialMonitor

### 项目类型
桌面应用程序 - 基于WebUI的串口调试工具

### 核心功能
一款功能齐全的串口调试助手，通过内置Web服务器提供现代化的浏览器界面，支持串口通信、数据收发、多种报文格式显示，并具备配置文件自动保存/加载能力。

### 目标用户
- 嵌入式开发工程师
- 硬件调试人员
- IoT开发者
- 电子爱好者

---

## 2. 技术架构

### 整体架构
```
┌─────────────────────────────────────────────────────────┐
│                    C++ Backend                          │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────┐  │
│  │ Serial Port │  │   Config    │  │   Web Server    │  │
│  │   Manager   │  │   Manager   │  │   (HTTP/WS)     │  │
│  └─────────────┘  └─────────────┘  └─────────────────┘  │
└───────────────────────────┬─────────────────────────────┘
                            │
                     HTTP/WebSocket

┌───────────────────────────┴─────────────────────────────┐
│                      Web Frontend                        │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────┐   │
│  │ Connection  │  │   Data     │  │   Settings &    │   │
│  │   Panel     │  │   Display  │  │   Themes        │   │
│  └─────────────┘  └─────────────┘  └─────────────────┘   │
└─────────────────────────────────────────────────────────┘
```

### 后端技术栈
- **编程语言**: C++17
- **串口通信**: Boost.Asio (跨平台支持)
- **HTTP服务器**: CivetWeb (轻量级嵌入式HTTP服务器)
- **WebSocket**: CivetWeb内置支持
- **配置文件**: JSON (nlohmann/json)
- **打包工具**: CMake + CPack

### 前端技术栈
- **框架**: Vue 3 + Vite
- **UI库**: Element Plus
- **状态管理**: Pinia
- **样式**: SCSS + CSS Variables

## 3. 功能规格

### 3.1 串口连接管理

#### 端口选择
- 自动扫描可用串口
- 支持刷新串口列表
- 显示串口详细信息（描述、VID/PID）

#### 波特率选择
- 标准波特率: 300, 1200, 2400, 4800, 9600, 19200, 38400, 57600, 115200, 230400, 460800, 921600
- 自定义波特率输入

#### 数据位配置
- 选项: 5, 6, 7, 8 (默认: 8)

#### 校验位配置
- 选项: None, Odd, Even, Mark, Space (默认: None)

#### 停止位配置
- 选项: 1, 1.5, 2 (默认: 1)

#### 流控制配置
- 选项: None, Hardware (RTS/CTS), Software (XON/XOFF) (默认: None)

#### 连接控制
- 连接/断开按钮
- 连接状态指示
- 自动重连选项

### 3.2 数据收发

#### 发送功能
- 文本输入框
- Hex输入模式
- 发送按钮
- 发送周期设置（定时发送）
- 行尾符选择 (CR, LF, CR+LF, None)

#### 接收功能
- 实时数据显示
- 时间戳显示（可选）
- 自动滚动
- 清空接收区

### 3.3 报文查看格式

#### 接收数据显示模式
- **ASCII**: 纯ASCII字符显示
- **Hex**: 十六进制显示 (如: `68 65 6C 6C 6F`)
- **UTF-8**: UTF-8解码显示
- **Decimal**: 十进制显示
- **Binary**: 二进制显示
- **混合模式**: Hex + ASCII对照

#### 字符编码
- 支持多种字符编码: UTF-8, GBK, GB2312, ASCII

### 3.4 高级功能

#### 数据记录
- 保存收发数据到文件
- 支持自动命名和手动命名
- 文件格式: .txt, .log, .hex

#### 数据过滤器
- 按关键字过滤
- 正则表达式支持
- 过滤颜色高亮

#### 搜索功能
- 接收区搜索
- 高亮匹配

#### 快捷命令
- 保存常用命令
- 快速发送

### 3.5 配置文件管理

#### 配置内容
```json
{
  "connection": {
    "port": "COM3",
    "baudrate": 115200,
    "databits": 8,
    "parity": "none",
    "stopbits": 1,
    "flowcontrol": "none"
  },
  "display": {
    "format": "hex",
    "encoding": "utf-8",
    "timestamp": true,
    "autoscroll": true
  },
  "sending": {
    "lineEnding": "crlf",
    "hexMode": false,
    "repeatEnabled": false,
    "repeatInterval": 1000
  },
  "ui": {
    "theme": "dark-blue",
    "language": "zh-CN"
  }
}
```

#### 配置保存
- 退出时自动保存
- 手动保存按钮
- 保存位置: 用户配置目录

### 3.6 UI主题系统

#### 内置主题
1. **深空蓝 (Dark Blue)** - 默认主题
2. **碳岩黑 (Carbon Black)** - 纯黑高端
3. **极客绿 (Matrix Green)** - 黑客风格
4. **日间白 (Daylight White)** - 明亮主题
5. **紫罗兰 (Violet)** - 紫色调
6. **日落橙 (Sunset Orange)** - 暖色调

#### 主题切换
- 下拉菜单选择
- 实时预览
- 切换动画 (300ms ease)

### 3.7 动画系统

#### 动画原则
- 流畅精致但不华丽
- 过渡时间: 150-300ms
- 使用CSS transitions和Vue transition

#### 动画场景
- 主题切换: 全局渐变过渡
- 面板展开/收起: slide-fade
- 按钮交互: scale + 颜色变化
- 连接状态: pulse动画
- 数据接收: subtle flash

## 6. 验收标准

### 功能验收
- [ ] 能正确扫描并列出所有可用串口
- [ ] 能使用配置的波特率打开串口
- [ ] 能发送文本和Hex数据
- [ ] 能接收并正确显示数据（支持多种格式）
- [ ] 配置文件能正确保存和加载
- [ ] 主题切换流畅无卡顿

### 性能验收
- [ ] 启动时间 < 3秒
- [ ] 数据接收延迟 < 10ms
- [ ] 界面切换动画流畅 (60fps)

### 打包验收
- [ ] 生成单个可执行exe文件
- [ ] 无需安装可直接运行
- [ ] 启动显示ASCII字符画
